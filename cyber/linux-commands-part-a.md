### Filesystem

List of commands
```bash
$ pwd
$ cd
$ ls
$ mkdir, rmdir

$ updatedb
$ passwd
$ man
```


### Network

```bash
$ sudo apt install net-tools
```
List of commands
```bash
$ ifconfig
$ iwconfig
$ ping
$ arp
$ netstat
$ route
```

### User management

List of commands
```bash
$ chmod
$ adduser
$ /etc/passwd, /etc/shadow
$ su
$ sudo
```
